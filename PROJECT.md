- **Core Loop:** HTML + Webhooks + Automated Web Service = Rendered UI.
- **Model Role:** Gemma-4-E2B (via OpenRouter) acts as a structured JSON code generator.

- `index.html`: Fully responsive chat interface on the left, sandboxed dynamic `<iframe>` preview pane on the right.

- `client.js`: Fetches user prompts, calls OpenRouter securely (or routes through a lightweight worker), parses Gemma's JSON output, and feeds the `<iframe>` dynamically via Blob URLs.

## 3. Security Requirements
- All desktop file synchronization actions must require an `x-api-key` header match.
